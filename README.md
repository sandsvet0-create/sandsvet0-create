# Psychiatric Assessment Platform

A secure web platform designed to automate psychometric assessment, support longitudinal monitoring of patients, and improve continuity of mental healthcare through digital technologies.

## 📖 About the Project

This project was developed as a digital health solution to simplify the use of standardized psychometric questionnaires in psychiatric practice. The platform automates score calculation, stores assessment history, and provides healthcare professionals with convenient tools for monitoring patient progress over time.

The system is based on role-based access control and includes separate interfaces for patients, physicians, and administrators. Particular attention has been paid to the protection of sensitive medical information through encryption and secure authentication mechanisms.

## ✨ Features

### 👤 Patient
- Complete standardized psychometric questionnaires
- Automatic score calculation and interpretation
- View previous assessment results
- Monitor symptom dynamics through charts
- Receive reminders for repeated assessments

### 👨‍⚕️ Physician
- Access patient assessment history
- Monitor changes over time
- Add clinical notes and comments
- Filter patients
- Automatically identify patients with worsening indicators
- Export reports to PDF

### 🛠 Administrator
- User management
- Role assignment
- Patient–physician linking
- Access control
- Backup support (planned)
- Audit log (planned)

## 🔒 Security

The platform was designed to handle sensitive medical information securely.

Implemented security measures include:

- Field-level encryption for sensitive database data
- bcrypt password hashing
- JWT authentication
- Role-based authorization
- Secure session management
- Compliance-oriented approach to personal data protection
